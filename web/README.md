# Olih Web

React web client for Olih.

## Stack

- React
- TypeScript
- Ant Design
- Create React App

## Working Directory

Run all frontend commands from the `web` directory.

## Install and Run

```bash
npm install
npm start
```

## Available Scripts

```bash
npm start
npm test
npm run build
```

## Notes

- The current package name is still `olih-demo`.
- The frontend includes local mock data for development and UI testing.
